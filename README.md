starfield
=========

0. Make sure you have ```ruby-2.0.0-p247```
1. ```gem install gosu```
2. ```ruby starfield.rb```
3. Use LEFT, RIGHT, and UP arrows to propel the ship through space
4. Use the COMMA and PERIOD keys to tune the radio
5. Find what you've been looking for
